## Common Commands

Since this is a learning repository, common Node.js commands will be used:

```bash
# Initialize a new Node.js project
npm init -y

# Install dependencies
npm install <package>

# Run a JavaScript file
node <filename>.js

# Run with npm script
npm run <script>
```
